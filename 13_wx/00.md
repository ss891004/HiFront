+ 结构赋值
```text
所谓的解构赋值，就是把 某个对象中的属性，当作变量，给解放出来，这样，今后就能够当作变量直接使用了

可以使用 : 为解构出来的变量重命名
const person = {
    name: 'zs',
    age: 22,
    gender: '男'
}
// 变量的解构赋值
let{ name: name123, age, gender } = person
name123 = '哈哈'
console.log(name123)//输出为123

```
